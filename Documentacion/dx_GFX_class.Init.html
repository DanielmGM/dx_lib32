Init

Inicializa el modo de video y las funciones graficas. 

 Public Function Init(
	  hwnd As Long,
	  Width As Long,
	  Height As Long,
	  Optional Bpp As Long = 16,
	  Optional Windowed As Boolean,
	  Optional TripleBuffer As Boolean,
	  Optional VSync As Boolean
	  Optional RefreshRate As Long = 60 ) As Boolean

Argumentos

hwnd
 Long. Identificador de la ventana.
Width
 Long. Ancho en píxeles del modo de video.
Height
 Long. Alto en píxeles del modo de video.
Bpp
 Optional. Long. Profundidad de color del modo de video.

Default: 16

Windowed
 Optional. Boolean. Indica si se inicializa en modo ventana.
TripleBuffer
 Optional. Boolean. Indica si se inicializa con Triple Buffer activado.
VSync
 Optional. Boolean. Indica si se inicializa con la espera de refresco vertical activada.
RefreshRate
 Optional. Long. Refresco del modo de video en hertzios. Si se pasa como valor 0 se usa el valor por defecto del sistema.

Default: 60

Resultado

Boolean -  Devuelve Verdadero en caso de inicializarse correctamente.

Comentarios

Inicializa el modo de video y los subsistemas de la clase habilitando las rutinas y la memoria para trabajar con gráficos.

En modo ventana se puede establecer cualquier resolución excepto la profundidad de color y la frecuencia de refresco que se tomara el valor que use el escritorio. En modo a pantalla completa solo se aceptan los modos de video soportados por el dispositivo de video.
