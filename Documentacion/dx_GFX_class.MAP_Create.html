MAP_Create

Crea un grafico vació en memoria. 

 Public Function MAP_Create(
	  Width As Long,
	  Height As Long ) As Long

Argumentos

Width
Long. Anchura en píxeles.
Height
Long. Altura en píxeles.

Resultado

Long -  Devuelve el código identificador para el grafico en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede crear la superficie se devuelve GFX_NOTCREATED.

Comentarios

Se pueden crear tantas superficies como se deseen teniendo como limite la memoria del dispositivo grafico.
