dx_GFX_class: Display_Mode

Display_Mode

Define los valores de un modo de video. 

 Public Type Display_Mode

Argumentos

Height
Long. Define la anchura en píxeles.
Width
Long. Define la altura en píxeles.
Bpp
Long. Define la profundidad de color en bits.
Refresh_Rate
Long. Define la frecuencia de refresco del monitor.

Comentarios

Esta estructura se utiliza para leer e interpretar los valores de un modo de video.
El campo Bpp siempre devolverá o 16 bits o 32 bits.
