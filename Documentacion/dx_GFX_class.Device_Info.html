dx_GFX_class: Device_Info

Device_Info

Define los valores de un dispositivo de video. 

 Public Type Device_Info

Argumentos

AdapterName
String. Nombre del dispositivo.
DriverName
String. Nombre del controlador de video.
DriverVersionHigh
Long. Numero de versión del controlador de video.
DriverVersionLow
Long. Numero de subversión del controlador de video.
Revision
Long. Numero de revisión del controlador de video. Corresponde al 3º grupo de números de la versión. E.j.: Revisión 918 de la version 2.0 = 2.0.918
VendorID
Long. Código del proveedor.
MaxTextureHeight
Long. Altura máxima de una textura en memoria para este dispositivo.
MaxTextureWidth
Long. Anchura máxima de una textura en memoria para este dispositivo.
MaxBitsPerPixel
Long. Profundidad máxima de color (o bits por pixel) para este dispositivo.

Comentarios

Esta estructura se utiliza para leer e interpretar los valores de un dispositivo de video.
