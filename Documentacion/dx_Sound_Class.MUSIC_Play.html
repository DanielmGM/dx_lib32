dx_Sound_Class: MUSIC_Play

MUSIC_Play

Reproduce una secuencia de música. 

 Public Function MUSIC_Play(
	  Music As Long,
	  Optional Buffer As Sound_Buffer = Primary_Buffer,
	  Optional Looping As Boolean ) As Boolean

Argumentos

Music
Long. Identificador de la secuencia de música.
Buffer
Optional. Sound_Buffer. Buffer donde se reproducirá la secuencia de música. Por defecto se usa el buffer primario.

Default: Primary_Buffer

Looping
Optional. Boolean. Indica si se reproducirá la secuencia de música en bucle.

Resultado

Boolean -  Devuelve Verdadero si la operación se realiza con éxito.

Comentarios

Las funciones de secuencias de música están diseñadas para dar soporte a formatos avanzados de audio e implementado para dar posibilidad de reproducir una secuencia de música de fondo. Se permite reproducir dos secuencias de música como máximo, permitiendo así poder hacer efectos de transición entre dos secuencias de música (efecto 'Fade') por ejemplo.
