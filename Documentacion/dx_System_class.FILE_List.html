dx_System_class: FILE_List

FILE_List

Lista los archivos y subdirectorios contenidos en una ruta especifica. 

 Public Function FILE_List(
	  List() As String,
	  Optional Path As String,
	  Optional Attrib As File_Attribute,
	  Optional Sort As Boolean,
	  Optional SortMode As Sort_Mode ) As Long

Argumentos

List
String. Parámetro de salida que devuelve una lista con los archivos y subdirectorios contenidos en un directorio especifico.
Path
Optional. String. Ruta del directorio que leeremos. Se pueden usar comodines para filtrar extensiones, por ejemplo "C:\Windows\*.*", "C:\Windows\*.bmp"...
Attrib
Optional. File_Attribute. Constante que define el atributo de los archivos que se listaran.
Sort
Optional. Boolean. Indica si se ordenan los elementos de la lista.
SortMode
Optional. Sort_Mode. Constante que define el modo de ordenación.

Resultado

Long -  Devuelve SYS_OK si la operación se ha realizado con éxito, SYS_EMPTYLIST si el directorio no contiene archivos y SYS_UNKNOWNERROR en caso de ocurrir cualquier otro error.
