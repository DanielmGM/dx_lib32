dx_GFX_class: Vertex

Vertex

Define los valores de un vértice 2D. 

 Public Type Vertex

Argumentos

X
Long. Coordenada X. Define la posición en el eje horizontal.
Y
Long. Coordenada Y. Define la posición en el eje vertical.
Z
Long. Coordenada Z. Define la profundidad de dibujo. Para saber como funciona este parámetro leer información acerca del ZBuffer.
Color
Long. Color ARGB que define el color del vértice.

Comentarios

Algunas funciones de la clase grafica necesitan definir valores de los vértices para el dibujo de ciertos gráficos.
