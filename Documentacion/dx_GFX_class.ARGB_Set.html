ARGB_Set

Genera un color ARGB a partir de los valores definidos en sus canales. 

 Public Function ARGB_Set(
	  Alpha As Integer,
	  Red As Integer,
	  Green As Integer,
	  Blue As Integer ) As Long

Argumentos

Alpha
Integer. Valor para el canal Alpha.
Red
Integer. Valor para el canal Rojo.
Green
Integer. Valor para el canal Verde.
Blue
Integer. Valor para el canal Azul.

Resultado

Long -  Devuelve un valor ARGB.

Comentarios

Esta función ayuda en la tarea de definir valores para los colores pudiendo generar cualquier gama tanto en 16 como en 32 bits color. La función es independiente de la profundidad de color establecida para el modo de video transformando el valor final en uno valido para el modo establecido. También se pueden definir valore ARGB directamente en formato Hexadecimal, como por ejemplo el color ARGB definido según el orden de sus canales 255, 62, 189 y 255 seria en Hexadecimal &HFF3EBDFF.
