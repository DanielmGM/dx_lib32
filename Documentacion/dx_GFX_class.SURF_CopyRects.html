SURF_CopyRects

Copia una serie de regiones de una superficie en otra. 

 Public Function SURF_CopyRects(
	  SrcSurf As Long,
	  DestSurf As Long,
	  SrcRect() As GFX_Rect,
	  DestPixel() As Vertex ) As Boolean

Argumentos

SrcSurf
Long. Identificador de la superficie origen.
DestSurf
Long. Identificador de la superficie destino.
SrcRect
GFX_Rect. Lista de regiones que se copiaran de la superficie origen a la superficie destino. El argumento se debe pasar apuntando al primer vector del array, el 0.
DestPixel
Vertex. Lista de coordenadas donde se copiaran las regiones en la superficie destino. El argumento se debe pasar apuntando al primer vector del array, el 0.

Resultado

Boolean -  Devuelve Verdadero si la operación se ha realizado con éxito.
