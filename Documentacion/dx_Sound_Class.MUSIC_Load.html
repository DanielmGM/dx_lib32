dx_Sound_Class: MUSIC_Load

MUSIC_Load

Carga un archivo de audio desde disco como una secuencia musical. 

 Public Function MUSIC_Load(
	  Filename As String ) As Long

Argumentos

Filename
String. Nombre y ruta del archivo de música.

Resultado

Long -  Devuelve el código identificador para la música en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede cargar la música se devuelve SND_FILENOTLOAD.

Comentarios

Esta función soporta de forma nativa formatos *.MIDI, *.WAV y *.MP3. Se pueden procesar otros formatos de audio como el *.WMA o el *.OGG si se instalan los codecs necesarios para DirectShow. Se pueden cargar tantas secuencias de música como se deseen teniendo como limite la memoria del sistema.
