dx_System_class: SYS_ScreenCapture

SYS_ScreenCapture

Realiza una captura de la pantalla o de la ventana que tiene el foco y la guarda en un archivo de formato Mapa de Bits de Windows (*.BMP). 

 Public Function SYS_ScreenCapture(
	  Filename As String,
	  Mode As Capture_Mode ) As Boolean

Argumentos

Filename
String. Nombre y ruta del archivo que se creara.
Mode
Capture_Mode. Modo de captura.

Resultado

Boolean -  Devuelve Verdadero si la operación se ha realizado con éxito.

Comentarios

Esta función no funciona correctamente con la clase grafica. Si desea hacer una captura de pantalla de un programa ejecutando la clase grafica dx_GFX deberá llamar a la función apropiada en la clase, SURF_ScreenCapture.
