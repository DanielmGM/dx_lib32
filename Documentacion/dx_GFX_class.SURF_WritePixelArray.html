SURF_WritePixelArray

Escribe el array de píxeles de una lista de pixeles existente.

 Public Function SURF_WritePixelArray(
	  Id As Long
	  pxArr() As Long ) As Boolean

Argumentos

Id
Long. Identificador de la lista de píxeles de la superficie.
pxArr
Long. Array de enteros de 32 bits que representan los valores ARGB de los pixeles.

Resultado

Boolean -  Devuelve Falso si array contiene un numero diferente de pixeles que la lista original.

Comentarios

Utilice este metodo para modificar los valores de los pixeles de una superficie de forma rapida y directa.
