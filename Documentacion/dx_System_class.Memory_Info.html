Memory_Info

Estructura que almacena las cantidades de memoria del sistema. 

 Public Type Memory_Info

Argumentos

Long. Cantidad total de memoria física.
AvailPhys
Long. Cantidad disponible de memoria física.
TotalPageFile
Long. Cantidad total de memoria de archivo de pagina.
AvailPageFile
Long. Cantidad disponible de memoria de archivo de pagina.
TotalVirtual
Long. Cantidad total de memoria virtual.
AvailVirtual
Long. Cantidad disponible de memoria virtual.

Comentarios

Esta estructura se utiliza como valor de retorno de la función SYS_GetMemory.
