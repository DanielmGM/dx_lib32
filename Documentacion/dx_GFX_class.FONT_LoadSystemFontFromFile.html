dx_GFX_class: FONT_LoadSystemFontFromFile

FONT_LoadSystemFontFromFile

Carga una fuente de texto TrueType desde un archivo y la prepara para ser usada con DRAW_Text. 

 Public Function FONT_LoadSystemFontFromFile(
	  Filename As String,
	  FontName As String,
	  Size As Long,
	  Bold As Boolean,
	  Italic As Boolean,
	  Underline As Boolean,
	  Strikethrough As Boolean ) As Long

Argumentos

Filename
String. Nombre del archivo que contiene la fuente de texto True Type.
FontName
String. Devuelve el nombre de la fuente de texto.
Size
Long. Tamaño con el se cargara la fuente.
Bold
Boolean. Indica si se cargara la fuente con estilo Negrita.
Italic
Boolean. Indica si se cargara la fuente con estilo Cursiva.
Underline
Boolean. Indica si se cargara la fuente con estilo Subrayado.
Strikethrough
Boolean. Indica si se cargara la fuente con estilo Tachado.

Resultado

Long -  Devuelve el código identificador para esta fuente en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede cargar la fuente se devuelve GFX_FILENOTLOAD.

Comentarios

La fuente de texto puede estar ubicada en cualquier directorio, no necesariamente el directorio de fuentes de windows. Se pueden cargar tantas fuentes de texto como se deseen teniendo como limite la memoria del sistema.
