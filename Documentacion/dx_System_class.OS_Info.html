dx_System_class: OS_Info

OS_Info

Estructura que almacena la información del sistema operativo. 

 Public Type OS_Info

Argumentos

MajorVersion
Long. Numero de la versión del sistema operativo.
MinorVersion
Long. Numero de la subversión del sistema operativo.
BuildNumber
Long. Numero de compilación.
PlatformId
OS_Plataform. Identificador de plataforma. Indica si la plataforma esta basada en Windows 9x (95/98/Me) o Windows NT (NT, 2000, XP, 2003).
WindowsId
String. Cadena de texto que devuelve el nombre y edición de la plataforma. Por ejemplo: "Microsoft Windows XP"

Comentarios

Esta estructura se utiliza como valor de retorno de la función SYS_GetOSInfo.
