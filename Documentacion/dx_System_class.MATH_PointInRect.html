MATH_PointInRect

Calcula si las coordenadas de un punto se encuentran dentro del área de un rectángulo. 

 Public Function MATH_PointInRect(
	  X As Long,
	  Y As Long,
	  DestRect As GFX_Rect ) As Boolean

Argumentos

X
Long. Coordenada X del punto.
Y
Long. Coordenada Y del punto.
DestRect
GFX_Rect. Rectángulo de destino.

Resultado

Boolean -  Devuelve Verdadero si las coordenadas del punto están dentro del area del rectángulo.

Comentarios

Esta función comprueba si un punto esta dentro del área de un rectangulo. Esta función puede ser útil para implementar colisiones entre objetos o gráficos.
