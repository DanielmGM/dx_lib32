dx_Input_class: Joystick_Data

Joystick_Data

Información de retorno del joystick o gamepad. 

 Public Type Joystick_Data

Argumentos

Button(1 To 12)
Boolean. Estado de los 12 botones.
X
Long. Coordenada X. Define la posición en el eje horizontal.
Y
Long. Coordenada Y. Define la posición en el eje vertical.

Comentarios

Esta estructura es la que se usa como retorno en el objeto Joystick.

Aviso programadores de Visual Basic .NET

Desde Visual Basic .NET esta estructura posiblemente falle en algunas maquinas donde retorne valores erróneos. El autor recomienda el siguiente componente para .NET que permite conectar con las nuevas interfaces XInput de DirectX9 y acceder a nuevas propiedades de los joysticks y gamepads actuales como el gamepad de Xbox 360 por ejemplo: MDXinfo.com - A simple XInput wrapper library
