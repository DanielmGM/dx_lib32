SOUND_Play

Reproduce un efecto de sonido. 

 Public Function SOUND_Play(
	  Sound As Long,
	  Optional Channel As Long = - 1,
	  Optional Looping As Boolean = False ) As Long

Argumentos

Sound
Long. Identificador del efecto de sonido.
Channel
Optional. Long. Canal donde se reproducirá el efecto de sonido. Por defecto se usa el primer canal libre que se encuentre.

Default: - 1

Looping
Optional. Boolean. Indica si se reproducirá el efecto de sonido en bucle.  

Default: False

Resultado

Long -  Devuelve el canal donde se reproducirá el sonido. Esta valor es necesario al hora de modificar parámetros en la reproducción de un sonido, para ponerlo en pausa o detenerlo. En caso de no poder reproducir el efecto de sonido devuelve -1.
