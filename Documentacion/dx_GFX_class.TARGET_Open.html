TARGET_Open

Configura un Render Target como buffer de dibujo. 

 Public Function TARGET_Open(
	  Target As Long )

Argumentos

Target
Long. Render Target destino para las operaciones de dibujo.

Resultado

Comentarios

Todas las llamadas de operaciones de dibujo apuntaran al grafico asociado del Render Target en vez de al back buffer. Utilice TARGET_Close para configurar el back buffer de nuevo como destino de las operaciones de dibujo. Solo se puede abrir un Render Target a la vez.
