dx_System_class: SYS_LockWINKeys

SYS_LockWINKeys

Desactiva el juego de combinaciones de teclas especiales de Windows. 

 Public Sub SYS_LockWINKeys(
	  Locked As Boolean )

Argumentos

Locked
Boolean. Desactiva o no las combinaciones de teclas especiales de Windows.

Comentarios

Esta función desactiva el juego de combinaciones de teclas especiales de Windows, tales como por ejemplo Ctrl+Alt+Supr, Ctrl+Esc, Alt+Tab, Logo Windows, tecla de menú contextual, etc...

Importante: Esta función solo esta soportada en las plataformas Windows 9x (95/98/Me). En plataformas NT no hace efecto alguno.
