dx_System_class: PAK_IsPAKFormat

PAK_IsPAKFormat

Comprueba que se trata de un archivo con formato PAK. 

 Public Function PAK_IsPAKFormat(
	  Filename As String ) As Boolean

Argumentos

Filename
String. Nombre y ruta del archivo PAK.

Resultado

Boolean -  Devuelve Verdadero si formato del archivo coincide con el formato sin compresión estándar PAK.

Comentarios

No es necesario que compruebe el formato de un archivo que va a ser cargado con PAK_Load ya que dicha comprobación se realiza automáticamente. Esta función solo esta pensada para realizar comprobaciones en otras acciones que no sean la carga de un archivo PAK.
