Archivos PAK


Un archivo PAK es un formato libre de paquete contendor de recursos que fue diseñado por ID Software para sus motores de sus juegos Quake®, Quake 2® y Hexen 2®, y es usado también por Valve Software en su motor para el juego Half-Life®. Este formato a diferencia de otros como el ZIP y el RAR no aplica compresión alguna a los archivos contenidos en el.

Los archivos PAK se pueden crear y editar mediante la utilidad gratuita PAKExplorer.

Las funciones implementadas para trabajar con este formato son: PAK_Load, PAK_IsPAKFormat, PAK_ExtractFile y PAK_ExtractFileInMemory.
