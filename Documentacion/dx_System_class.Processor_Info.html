dx_System_class: Processor_Info

Processor_Info

Estructura que almacena la información del procesador principal. 

 Public Type Processor_Info

Argumentos

Nombre
String. Nombre del procesador.
Identifier
String. Identificador del modelo y familia a la que pertenece.
VendorIdentifier
String. Nombre del fabricante.
Mhz
Integer. Megahercios del procesador. En plataformas Windows 9x (95/98/Me) esta variable devuelve 0.

Comentarios

Esta estructura se utiliza como valor de retorno en la función SYS_GetProcessorInfo.
