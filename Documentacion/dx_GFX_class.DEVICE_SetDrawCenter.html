DEVICE_SetDrawCenter

Configura una nueva posicion de dibujo en una operacion de dibujo. 

 Public Function DEVICE_SetDrawCenter(
	  X As Long,
	  Y As Long )

Argumentos

X
Long. Nuevo origen de dibujo para X.
Y
Long. Nuevo origen de dibujo para Y.

Resultado

Comentarios

Este procedimiento permite definir un nuevo origen de dibujo y eje de rotacion en una operacion de dibujo de imagenes.
Dichos valores se reiniciaran despues de terminar una llamada a cualquiera de las funciones funciones a las que se aplica los valores de este procedimiento:
DRAW_Map, DRAW_MapEx.

Este metodo no afecta a DRAW_MapEx si el parametro Center es Verdadero.
