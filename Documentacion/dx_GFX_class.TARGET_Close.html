TARGET_Close

Cierra el Render Target activo y configura de nuevo el back buffer como buffer de dibujo.

Resultado

Comentarios

Una vez cerrado el Render Target el grafico asociado esta listo para ser dibujado sobre el back buffer. Se puede volver a abrir el Render Target y seguir dibujando sobre el grafico asociado llamando antes a procedimiento TARGET_Open.
