TARGET_Create

Crea un Render Target y lo asocia a un grafico. 

 Public Sub TARGET_Create(
	  Map As Long )

Argumentos

Map
Long. Grafico que se utilizara de destino para las operaciones de dibujo.

Resultado

Long - Devuelve el código identificador del Render Target creado, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede crear la superficie se devuelve GFX_NOTCREATED.

Comentarios

Se pueden crear tantos Render Targets como se deseen teniendo como limite la memoria del dispositivo grafico.
