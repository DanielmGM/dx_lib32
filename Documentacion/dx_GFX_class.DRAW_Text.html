DRAW_Text

Dibuja una cadena de texto con formato en pantalla. 

 Public Sub DRAW_Text(
	  Font As Long,
	  Text As String,
	  X As Long,
	  Y As Long,
	  Z As Long,
	  Color As Long,
	  Align As Text_Align )

Argumentos

Font
Long. Identificador de la fuente.
Text
String. Cadena de texto que se va a dibujar en pantalla.
X
Long. Coordenada horizontal de dibujo. Este valor no altera la posicion de dibujo del texto cuando el valor del parámetro Align es Align_Center.
Y
Long. Coordenada vertical de dibujo.
Z
Long. Coordenada de profundidad de dibujo. Para saber como funciona este parámetro leer información acerca del ZBuffer.
Color
Long. Color ARGB que se aplicara para realizar la operación de dibujo.
Align
Text_Align. Define el alineamiento del texto en pantalla.

Comentarios

Para usar esta función previamente se debe haber cargado una fuente en memoria mediante FONT_LoadSystemFont.

Esta función es lenta por lo que no conviene abusar de ella ya que el rendimiento de nuestro programa se reducirá considerablemente. Para tratar de lograr un poco de velocidad a la hora de dibujar cadenas de texto estáticas, que no cambiaran su contenido en mitad de ejecución, dibuje el texto sobre un grafico mediante un Render Target.
