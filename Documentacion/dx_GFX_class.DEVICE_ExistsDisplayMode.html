DEVICE_ExistsDisplayMode

Comprueba que la resolución indicada es soportada por el dispositivo de video. 

 Public Function DEVICE_ExistsDisplayMode(
	  Width As Long,
	  Height As Long,
	  Optional Bpp As Long = 16,
	  Optional RefreshRate As Long = 60 ) As Boolean

Argumentos

Width
Long. Anchura del modo de video.
Height
Long. Altura del modo de video.
Bpp
Optional. Long. Profundidad de color.

Default: 16

RefreshRate
Optional. Long. Refresco del modo de video en hertzios. Si se pasa como valor 0 se usa el valor por defecto del sistema.

Default: 60

Resultado

Boolean -  Devuelve Verdadero si el modo de video existe y es soportado por el dispositivo de video.

Comentarios

Esta función no requiere inicializar la clase para ser ejecutada.
