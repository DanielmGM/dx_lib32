Conceptos y definiciones: Color ARGB

Color ARGB


Un color ARGB es un valor de tipo entero largo (Long) que define un color de 32 bits en los que se definen los 3 canales que componen un color: Rojo, Verde y Azul, y un cuarto canal, Alfa, que define el nivel de opacidad del color.

El canal Alfa se utiliza en la mayoría de los casos como parámetro para indicar el nivel de opacidad al hora de realizar una operación grafica tal como pintar una primitiva o un sprite. Todos los canales comprenden un rango de 0 a 255, siendo 0 el valor mínimo de opacidad para el Alfa haciendo totalmente transparente el color, y 255 siendo el nivel máximo de opacidad, haciendo totalmente solidó el color.

Se pueden generar colores indicando el valor de cada canal bien mediante su equivalente en hexadecimal o bien mediante la función ARGB_Set.
