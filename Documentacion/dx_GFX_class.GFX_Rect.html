dx_GFX_class: GFX_Rect

GFX_Rect

Define los valores de un rectángulo. 

 Public Type GFX_Rect

Argumentos

X
Long. Coordenada X. Define la posición en el eje horizontal de la esquina superior izquierda.
Y
Long. Coordenada Y. Define la posición en el eje vertical de la esquina superior izquierda.
Width
Long. Define la anchura del rectángulo.
Height
Long. Define la altura del rectángulo.

Comentarios

Esta estructura se utiliza para delimitar zonas en algunas operaciones de dibujo con gráficos.
