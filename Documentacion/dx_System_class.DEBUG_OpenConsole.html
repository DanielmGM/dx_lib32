dx_System_class: DEBUG_OpenConsole

DEBUG_OpenConsole

Abre una consola de depuración. 

 Public Function DEBUG_OpenConsole(
	  Title As String,
	  Optional SetDefaultWndPos As Boolean = Verdadero ) As Long

Argumentos

Title
String. Titulo de la consola de depuración.
 
SetDefaultWndPos
Optional. Boolean. Indica si la ventana de la consola de depuración se creara en una posición por defecto o no.  

Default: Verdadero

Resultado

Long -  Instancia de la consola de depuración.

Comentarios

Esta consola de depuración no es mas que una ventana de comandos del sistema, una ventana de MS-DOS. La idea de usar esta consola de depuración no es otra que la de obtener una sencilla herramienta para poder volcar mensajes a la hora de depurar nuestro programa o para el uso que se desee.
