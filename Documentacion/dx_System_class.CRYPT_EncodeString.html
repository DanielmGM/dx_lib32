CRYPT_EncodeString

Codifica una cadena de texto. 

 Public Function CRYPT_EncodeString(
	  Text As String,
	  Key As String ) As String

Argumentos

Text
String. Cadena de texto que será codificada.
Key
String. Clave de codificación.

Resultado

String -  Devuelve la cadena de texto codificada.

Comentarios

Esta función codifica una cadena de texto acorde el valor de la clave de codificación. Dicha clave será necesaria para poder decodificar después la cadena de texto.
