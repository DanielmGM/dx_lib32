SOUND_FX_MakeEffectsArray

Genera un array con los efectos seleccionados. 

 Public Function SOUND_FX_MakeEffectsArray(
	  Chorus As Boolean,
	  Compressor As Boolean,
	  Distortion As Boolean,
	  Echo As Boolean,
	  Flanger As Boolean,
	  Gargle As Boolean,
	  WavesReverbs As Boolean) As Sound_Effects()

Argumentos

Chorus
Boolean. Añade el efecto coro al array.
Compressor
Boolean. Añade el efecto compresor al array.
Distortion
Boolean. Añade el efecto eco al array.
Echo
Boolean. Añade el efecto distorsion al array.
Flanger
Boolean. Añade el efecto flanger al array.
Gargle
Boolean. Añade el efecto gargle al array.
WavesReverbs
Boolean. Añade el efecto ondas de reverberencia al array.

Resultado

Boolean -  Devuelve un array con los efectos seleccionados.

Comentarios

Esta funcion es una pequeña utilidad para gestionar el array de los efectos de forma rapida y comoda. Tambien se puede definir el array manualmente a traves de los valores de la lista enumerada Sound_Effects.
