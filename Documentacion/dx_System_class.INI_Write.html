dx_System_class: INI_Write

INI_Write

Escribe una clave de un archivo de configuración INI. 

 Public Sub INI_Write(
	  Filename As String,
	  Key_Value As String,
	  Key_Name As String,
	  Value As String )

Argumentos

Filename
String. Nombre y ruta del archivo de configuración INI. Si el archivo no existe se crea automáticamente.
Key_Value
String. Valor de la sección donde se encuentra la clave. Si la clave no existe se crea automáticamente.
Key_Name
String. Nombre de la clave. Si la clave no existe se crea una automaticamente.
Value
String. Valor de la sección donde se encuentra la clave. Si la clave no existe se crea automáticamente.

Comentarios

No es preciso crear un archivo INI previamente, si este no existe, al igual que alguna sección o clave, se creara automáticamente con los valores pasados en los argumentos.
