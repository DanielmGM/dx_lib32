dx_System_class: DLG_SaveFile

DLG_SaveFile

Abre el cuadro de dialogo de guardar archivo de Windows. 

 Public Function DLG_SaveFile(
	  hwnd As Long,
	  Filter As String,
	  Title As String,
	  InitDir As String,
	  Optional Filename As String,
	  Optional FilterIndex As Long ) As String

Argumentos

hwnd
Long. Identificador de la ventana que hará de cliente. Si no existe ventana cliente este parámetro se establecerá a 0.
Filter
String. Cadena de texto que define el filtro de las extensiones de archivo que se mostraran. Por ejemplo: "Archivos de texto *.TXT|*.txt|Todos los archivos *.*|*.*". También se puede definir un mismo elemento del filtro para que reconozca varios tipos de extensiones, un ejemplo: "Todos los archivos de imagen|*.bmp;*.gif;*.jpg,*.png".
Title
String. Cadena de texto que define el titulo de la ventana de dialogo.
InitDir
String. Ruta del directorio que mostrara el explorador. Si no se especifica ruta se toma la ruta por defecto.
Filename
Optional. String. Nombre de archivo que se mostrara por defecto en el cuadro de texto Nombre.
FilterIndex
Optional. Long. Índice de la lista de filtros que indica la extensión de archivo por defecto que se mostrara en explorador.

Resultado

String -  Nombre y ruta del archivo establecido.

Comentarios

Los cuadros de dialogo de Windows son interfaces de ayuda que solo devuelven información, no realizan ninguna acción por el usuario como por ejemplo la escritura en archivos.
