dx_System_class: MATH_RandSeed

MATH_RandSeed

Establece una semilla para crear un numero aleatorio. 

 Public Sub MATH_RandSeed(
	  Seed As Single )

Argumentos

Seed
Single. Valor que se utilizara como semilla para el generador de numeros aleatorios.

Comentarios

El uso de MATH_RandSeed es similar a la instrucción Randomize de Visual Basic. La semilla generada con Randomize es compatible para la función MATH_Rand.
