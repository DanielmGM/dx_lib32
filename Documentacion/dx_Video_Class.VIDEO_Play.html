dx_Video_Class: VIDEO_Play

VIDEO_Play

Reproduce una secuencia de video. 

 Public Function VIDEO_Play(
	  Video As Long,
	  X As Long,
	  Y As Long,
	  Width As Long,
	  Height As Long ) As Boolean

Argumentos

Video
Long. Identificador de la secuencia de video.
X
Long. Coordenada horizontal de la pantalla donde se renderizara el video.
Y
Long. Coordenada vertical de la pantalla donde se renderizara el video.
Width
Long. Ancho con el que se renderizara el video.
Height
Long. Alto con el que se renderizara el video.

Resultado

Boolean -  Devuelve Verdadero si la operación se realiza con éxito.

Comentarios

Esta clase esta diseñada para poder reproducir videos para cinematicas o introducciones en el programa. Dado que la implementación del mismo no permite integrarse a la perfección con la clase grafica se recomienda alternar el uso de una clase con la otra y se limita por ello la posibilidad de reproducir solo 1 video a la vez. Es aconsejable a la hora de reproducir una secuencia de video no llamar a la función Frame de la clase grafica dx_GFX.
