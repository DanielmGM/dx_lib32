HITIMER_Create

Crea un cronometro de alta precisión. 

 Public Function HITIMER_Create ( ) As Long

Argumentos

Resultado

Long -  Devuelve el identificador del cronometro de alta precisión. Si no se puede crear el cronometro se devuelve SYS_NOTCREATED, si el sistema no tiene soporte para cronómetros de alta precisión se devolvera SYS_HITIMERNOTSUPPORT

Comentarios

Los cronómetros son útiles para medir tiempos y realizar acciones periódicas, como animaciones por ejemplo.
