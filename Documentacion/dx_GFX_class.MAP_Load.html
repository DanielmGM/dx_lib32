dx_GFX_class: MAP_Load

MAP_Load

Carga un archivo de imagen desde disco como un grafico. 

 Public Function MAP_Load(
	  Filename As String,
	  Optional ColorKey As Long
	  Optional Smooth As Boolean = Verdadero ) As Long

Argumentos

Filename
String. Nombre y ruta de la imagen.
ColorKey
Optional. Long. Valor que define el color transparente para este grafico.
Smooth
Optional. Boolean. Valor que define si el grafico se cargara con suavizado de píxeles o no.

Default: Verdadero

Resultado

Long -  Devuelve el código identificador para este grafico en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede cargar el grafico se devuelve GFX_FILENOTLOAD.

Comentarios

Esta función soporta los siguientes formatos gráficos: *.DDS, *.TGA, *.JPG, *.JPEG, *.JFIF, *.JPE, *.BMP, *.DIB, *.TIF, *.TIFF, *.PNG, *.PPM. Se pueden cargar tantos gráficos como se deseen teniendo como limite la memoria del dispositivo grafico.
