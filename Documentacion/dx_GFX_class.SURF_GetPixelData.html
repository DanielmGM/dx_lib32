dx_GFX_class: SURF_GetPixelData

SURF_GetPixelData

Crea una copia en memoria de la lista de los píxeles de una superficie que contiene el valor ARGB de cada píxel. 

 Public Function SURF_GetPixelData(
	  Surf As Long ) As Long

Argumentos

Surf
Long. Identificador de la superficie que vamos a leer.

Resultado

Long -  Devuelve el código identificador para esta lista de píxeles en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede copiar la lista de píxeles se devuelve GFX_DATANOTREAD.

Comentarios

Las copias realizadas con esta función sirven para poder leer y/o escribir los valores de los colores de los píxeles de una superficie. Se pueden realizar tantas copias como se precisen.
