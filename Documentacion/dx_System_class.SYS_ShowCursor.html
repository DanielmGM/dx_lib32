dx_System_class: SYS_ShowCursor

SYS_ShowCursor

Muestra u oculta el cursor del ratón que dibuja Windows. 

 Public Sub SYS_ShowCursor(
	  Show As Boolean )

Argumentos

Show
Boolean. Indica si se muestra o no el cursor del ratón de Windows.

Comentarios

Esta función simplemente desactiva la presentación del cursor del ratón de Windows sobre las ventanas del programa pero no deshabilita la lectura de las coordenadas del cursor.
